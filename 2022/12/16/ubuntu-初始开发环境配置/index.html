ubuntu 初始开发环境配置

ubuntu 20.04 初始化配置

当配置好ubuntu虚拟机后，我们需要对其进行配置，方面更好的开发。

换源

打开software&update中心，可以看到如下图所示

点击Download from下拉框选择Other，找到China，然后选择对应的源即可，如我选的是https://mirrors.aliyun.com/ubuntu/

代理设置

打开设置中的网络，点击Network Proxy，我们可以看到如下图所示

设置自己的主机IP地址和对应代理的端口

更新一下

git配置

  1. 安装 vim，git
  2. 配置用户名和邮箱
  3. 配置SSH 查看生成秘钥
  4. GitHub上配置SSH
    • 登录GitHub
    • 点击settings->SSH and GPS keys->New SSH key
    • 将其刚刚生成的公钥复制到ssh配置中即可。
  5. git配置代理 IP地址需要跟代理主机ip地址一样，端口号是代理端口号

SSH配置

密码登录配置

  1. 下载openssh-server
  2. 确定Ubuntu SSH服务状态
  3. 测试登录(win10)

免密登录配置

  1. 生成公钥文件（win10）
  2. 打开ubuntu中，复制刚刚生成公钥到~/.ssh/authorized_keys中
  3. 修改文件权限
  4. 修改SSH配置文件（ubuntu） 添加一下三项
  5. 重新启动SSH服务
  6. 测试免密登录(win10)

vscode 远程连接配置(win10)

  1. 下载vscode插件 Remote - SSH，如下图所示
  2. 下载完成后，可以看到左边侧边栏多了一个电脑的标签
  3. 点击设置，即可看到一个配置文件，填写对应的配置即可
  4. 配置完成后，测试连接即可
